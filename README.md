## Overview
This project was developed as part of the **IEEE Hackathon 2023** to analyze e-commerce business sales data and generate insights using an interactive dashboard.

The dashboard provides a comprehensive view of sales performance, profits, and returns across different regions, categories, and time periods.

---

## Business Problem
E-commerce businesses need to understand sales trends, customer behavior, and return patterns to improve profitability and decision-making.

This project helps to:
- Analyze monthly sales and profit trends  
- Identify top-performing countries and cities  
- Understand return patterns by category  
- Enable interactive filtering using slicers  

---

## Key Metrics & Analysis

### Sales Analysis
- Monthly sales trends show consistent performance with peak sales in specific months  
- The United States contributes the highest share of total sales  

### Profit Analysis
- Highest profits are generated from the United States  
- Other contributing countries include Australia, France, Mexico, and Germany  

### Returns Analysis
- Fashion category has the highest number of returns  
- Cities like New York, Los Angeles, and San Francisco show high return volumes

## Dashboard Features
- Interactive slicers for:
  - Ship Mode  
  - Region  
  - Product Category

---

## Business Insights
- Sales are highest during peak seasons (e.g., end of year)  
- Returns are concentrated in specific product categories  
- Certain cities and regions contribute significantly to sales and returns  
- Data-driven insights can improve inventory and marketing strategies
